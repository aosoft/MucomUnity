MUCOM88 for Unity
====

[mucom.NET](https://github.com/kuma4649/mucomDotNET) を Unity の AudioClip として使えるようにしたものです。


## 使い方

1. UniRx をインストール
2. StreamingAssets 下に PCM 、音色ファイル等を配置
3. Unity Editor 上で実行
